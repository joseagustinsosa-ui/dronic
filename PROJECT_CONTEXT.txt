Decisiones de negocio ya tomadas:
- roles:
  - admin
  - operator
  - client
- operator incluye socio y piloto
- socio NO será admin
- admin valida recepción final del dinero por DRONIC
- flujo real de dinero puede ser:
  cliente -> piloto/operator -> socio/operator -> DRONIC -> validación admin
- por eso no basta con un solo campo pagado sí/no
